# Pandit Home Wall Paint — Design Guidelines

## Aesthetic Stance
**Modern Geometric & Color-Confident** — Clean lines, bold color blocking, and asymmetric layouts that showcase the creative and transformative nature of painting services. The design uses strategic color zones to demonstrate the company's expertise with color and space.

## Typography
- **Display**: Work Sans (700-800 weight) — Bold, geometric, modern sans-serif for headings
- **Body**: Inter (400-500 weight) — Clean, readable humanist sans for body text
- **Accent**: DM Sans (500-600 weight) — Secondary sans for labels and UI elements

## Color Palette
Inspired by paint swatches and modern interiors:
- **Primary (Deep Navy)**: #1a2332 — Professional, trustworthy base
- **Accent (Warm Terra)**: #e67e50 — Energetic, creative, paint-inspired
- **Secondary (Sage)**: #7fa99b — Calm, natural, interior design
- **Highlight (Sand)**: #f4e8d8 — Warm, welcoming background
- **Neutral Light**: #f8f9fa — Clean, spacious
- **Neutral Dark**: #2d3748 — Grounding contrast

## Canvas Treatment
Two-tone asymmetric sections with bold color blocking. Each major section uses contrasting backgrounds to create visual rhythm and demonstrate color expertise. Geometric shapes and diagonal splits add dynamic energy.

## Layout Principles
1. Asymmetric grid with intentional white space
2. Bold section transitions with color blocks
3. Large, confident typography
4. Geometric accent elements (paint stroke inspired)
5. Responsive breakpoint at 1024px

## Component Patterns
- **Hero**: Full-height with diagonal color split, oversized heading
- **Services**: Card grid with hover states and icon integration
- **Projects**: Image showcase with overlay captions
- **Contact**: Clean form with accent color CTAs
- **Navigation**: Fixed header with smooth scroll

## Craft Details
- Subtle hover transitions (200-300ms)
- Focus states with visible rings
- Generous padding and breathing room
- Real project names and realistic content
- Accessible color contrast (AA minimum)
